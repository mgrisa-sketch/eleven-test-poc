# Eleven Group - Test POC

POC Next.js 15 pour tester la stack de développement Eleven Group.

## Stack
- **Framework**: Next.js 15 (16.2.7)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Deployment**: Vercel
- **Repository**: GitHub

## Quick Start

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build

# Start production server
npm start
```

## Déploiement sur Vercel

### Option 1: Via l'interface Vercel (Recommandé)
1. Visitez [vercel.com/new](https://vercel.com/new)
2. Importez le repository: `mgrisa-sketch/eleven-test-poc`
3. Vercel détectera automatiquement Next.js
4. Cliquez sur "Deploy"

### Option 2: Via CLI
```bash
# Login to Vercel
vercel login

# Deploy
vercel --prod
```
